# Esqueleto de projeto Vue 3 + TailwindCSS

## Setup do projeto
```
npm install
```

### Compilar
```
npm run dev
npm run tw:build
```

### Compilar para produção
```
npm run build
```

### Possui 
Vue Router
VueX
Axios
TailwindCSS
TypeScript
Babel
Vue Auto Import Statements
Vue Auto Import Components
